# EasyCalc

## What is EasyCalc ?
EasyCalc is a calculator developed using React.

## What's next ?
Many functionnalities are coming soon ! Like :
1. Theme manager to customize your own calculator !
2. Expert mode
3. Mobile support

## How can i run this *amazing* website ?
To install simply run : 
`npm install`

And to start run : 
`npm run dev`

*~ Et voila ~*

Enjoy !
